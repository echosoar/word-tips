Word Tips 扩展测试页面

说明：这个页面用于测试 Word Tips 浏览器扩展的功能。请先在扩展设置中配置词汇，然后刷新此页面查看效果。

技术词汇示例

在现代 Web 开发中，API 是不可或缺的一部分。开发者经常需要处理 JSON 数据格式， 并使用 HTML 来构建用户界面。React 是一个流行的前端框架，用于构建交互式的用户界面。

CSS 用于样式化网页，而 JavaScript 则负责添加动态功能。后端开发通常使用 Node.js、 Python 或 Java 等语言。数据库方面，开发者可能会选择 MySQL、MongoDB 或 PostgreSQL。

建议的测试词汇配置

在扩展设置页面的"本地词表"中输入以下内容：

API:Application Programming Interface，应用程序编程接口
JSON:JavaScript Object Notation，一种轻量级的数据交换格式
HTML:Hypertext Markup Language，超文本标记语言
CSS:Cascading Style Sheets，层叠样式表
JavaScript:一种高级的、解释型的编程语言
React:用于构建用户界面的 JavaScript 库
Node.js:基于 Chrome V8 引擎的 JavaScript 运行时
Python:一种高级编程语言，语法简洁清晰
MySQL:开源的关系型数据库管理系统
MongoDB:面向文档的 NoSQL 数据库

测试功能

  1. 词汇高亮：配置词汇后，页面上的相关词汇应该显示点状虚线
  2. 悬停提示：鼠标悬停在高亮词汇上应该显示释义浮层
  3. 多个释义：如果一个词有多个释义，它们应该用横线分隔
  4. 隐藏词汇：在浮层中点击"不再显示"可以隐藏该词汇
  5. 网站排除：在排除列表中添加此页面的 URL 应该禁用高亮功能

更多内容

在云计算时代，AWS、Azure 和 GCP 是三大主流云服务提供商。开发者可以使用这些平台 部署应用程序，使用 Docker 进行容器化，通过 Kubernetes 进行编排管理。

前端开发工具链包括 Webpack、Babel、ESLint 等。版本控制通常使用 Git，代码托管 在 GitHub、GitLab 或 Bitbucket 上。CI/CD 流程可以通过 Jenkins、Travis CI 或 GitHub Actions 实现自动化。
