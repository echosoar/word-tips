✨ 下划线样式改进

改进内容： 将词汇高亮的下划线从粗重的深色样式改为更轻盈、更柔和的样式，提升视觉体验。

视觉对比

修改前
在现代 Web 开发中，API 是不可或缺的一部分。开发者经常需要处理 JSON 数据格式，并使用 HTML 来构建用户界面。

❌ 2px 深色实线，视觉较重

修改后
在现代 Web 开发中，API 是不可或缺的一部分。开发者经常需要处理 JSON 数据格式，并使用 HTML 来构建用户界面。

✓ 1px 半透明，更轻盈柔和

技术细节

  • 线宽从 2px 减少到 1px
  • 颜色从 #1a73e8 (不透明) 改为 rgba(26, 115, 232, 0.5) (50% 透明度)
  • 悬停时透明度从完全不透明改为 80%
  • 保持点状虚线样式，更加优雅
CSS 变更：
border-bottom: 2px dotted #1a73e8;
↓
border-bottom: 1px dotted rgba(26, 115, 232, 0.5);
